# OSMI — Open-Source Microwave Imaging

A MATLAB framework for 2-D microwave imaging. OSMI provides a complete pipeline for forward electromagnetic (EM) simulations and microwave imaging (MWI) reconstructions.

## Forward solver

The forward solver is **fem2d**, a compiled 2-D finite-element solver for the Helmholtz equation under TE polarisation. It assembles and solves the system on a triangular mesh generated by MATLAB's PDE Toolbox, returning the electric field at every node and cell centroid.

Two scripts must be run in order:

### Step 1 — Generate background field

**File:** `Forward_solvers/generate_background_field.mlx`

Computes the incident (background) electric field for a **homogeneous** Domain of Interest (DOI) — no target contrast. The field is driven by each antenna in the array in turn. Results are saved to `results/background_fields.mat` and `results/geometry.mat`.

### Step 2 — Generate scattered field

**File:** `Forward_solvers/generate_scattered_field.mlx`

Computes both the incident field and the total field (DOI with embedded target) on a **new, independent mesh**. The scattered field is formed as:

```
Es = Et − Ei
```

Using a different mesh from Step 1 is essential to avoid the **inverse crime** — reusing the same discretisation for data generation and reconstruction produces artificially perfect results that do not reflect real experimental conditions. Results are saved to `results/scattered_fields.mat` and `results/geometry_sf.mat`.

---

## Inverse solver

### Step 3 — Born approximation / TSVD imaging

**File:** `Imaging/Born_aproximation_TSVD/imaging.mlx`

Reconstructs the complex dielectric contrast profile inside the DOI from the scattered field data. The pipeline:

1. Loads `background_fields.mat`, `geometry.mat`, and `scattered_fields.mat`
2. Builds the data operator **F_D** using the Born approximation and Green's function reciprocity
3. Decomposes F_D via the economy SVD
4. Inverts with TSVD (energy-based truncation threshold)
5. Displays the real and imaginary parts of the recovered permittivity contrast ε′ and ε″

---

## How to run

Open MATLAB and run the three Live Scripts in order:

```
1. Forward_solvers/generate_background_field.mlx
2. Forward_solvers/generate_scattered_field.mlx
3. Imaging/Born_aproximation_TSVD/imaging.mlx
```

Each script is self-contained and saves its outputs to `Forward_solvers/results/`. Step 3 reads from that folder automatically.

## Dependencies

- MATLAB R2023b or later
- PDE Toolbox (for mesh generation)
- `fem2d.p` — compiled FEM solver (included in `Forward_solvers/lib/`)
